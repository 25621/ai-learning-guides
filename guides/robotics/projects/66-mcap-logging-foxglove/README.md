# MCAP Logging + Foxglove

## Key Insight

Debugging complex, multi-rate robotics systems requires recording high-frequency data streams—such as images, point clouds, and transforms—without sacrificing performance. Using [MCAP](/shared/glossary/#mcap) as a lightweight, standardized container format allows efficient serialization and structured playback of these diverse messages. When integrated with [Foxglove](/shared/glossary/#foxglove), developers can visualize live or recorded robot trajectories, coordinate frames, and camera feeds to quickly pinpoint where hardware or software behaviors diverged.
