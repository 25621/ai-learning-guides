# Tensor Core Utilization

## Key Insight

Profiling matrix operations using [Nsight Systems](/shared/glossary/#nsight-systems) and [Nsight Compute](/shared/glossary/#nsight-compute) reveals the efficiency of [Tensor Cores](/shared/glossary/#tensor-core) on the [GPU](/shared/glossary/#gpu). By measuring what percentage of execution time is spent on these specialized units, developers can identify memory bottlenecks and design kernels that achieve peak arithmetic performance.
