# Spec Compare

## Key Insight

Comparing the specifications of consumer and datacenter [GPUs](/shared/glossary/#gpu) (such as [Hopper](/shared/glossary/#hopper) and [Blackwell](/shared/glossary/#blackwell)) highlights the trade-offs in peak floating-point operations and memory bandwidth. Calculating the [arithmetic intensity](/shared/glossary/#ai-arithmetic-intensity) break-even point for each chip helps developers determine whether a workload will be compute-bound or memory-bound on that platform.
