# Cliff Walking을 위한 SARSA 🏔️

## 무엇인가요?

가장자리에 **위험한 절벽**이 있는 **매우 긴 복도**를 상상해 보세요. 절벽에서 떨어지면 처음 시작점으로 되돌아가야 합니다! 여러분의 목표는 떨어지지 않고 한쪽 끝에서 다른 쪽 끝까지 최대한 빨리 걸어가는 것입니다.

**SARSA**는 연습을 통해 이 복도를 걷는 법을 배우는 로봇입니다. 이 로봇은 절벽을 피하는 *안전한 경로*를 택하는 법을 배웁니다. 설령 그 길이 조금 더 멀더라도 말이죠. 왜냐하면 로봇 스스로가 탐험 중에 실수로 미끄러져 절벽 근처로 갈 수도 있다는 사실을 알고 있기 때문입니다!

---

## 핵심 아이디어: 실제로 하는 행동으로부터 배우기

SARSA라는 이름은 각 단계의 정보를 나타냅니다: **S**tate(상태) → **A**ction(행동) → **R**eward(보상) → **S**tate(다음 상태) → **A**ction(다음 행동)

SARSA가 학습에 사용하는 5가지 정보는 다음과 같습니다.

1. **S** — 지금 나는 어디에 있는가? (현재 상태)
2. **A** — 내가 실제로 취한 행동은 무엇인가?
3. **R** — 내가 받은 보상은 얼마인가?
4. **S** — 내가 도착한 곳은 어디인가?
5. **A** — 내가 *다음에 실제로 취할 행동*은 무엇인가?

마지막 "A"가 SARSA를 특별하게 만드는 점입니다! SARSA는 (설령 그것이 무작위적인 탐험일지라도) 자신이 *다음에 실제로 할 행동*을 사용하여 가치를 업데이트합니다. 이상적인 완벽한 행동이 아니라요.

**실생활 예시:** 자전거 타기를 배운다고 생각해 보세요. 가끔 몸이 무작위로 흔들릴 수 있다는 것(탐험)을 안다면, 주차된 차들로부터 조금 더 거리를 두고 달릴 것입니다. 흔들리다가 차에 부딪힐 수도 있으니까요! SARSA가 바로 이 일을 합니다: 자신의 무작위 실수를 고려하여 안전한 경로를 배웁니다.

## Cliff Walking (절벽 걷기) 지도

```
[ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ]
[ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ]
[ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ]
[S][C][C][C][C][C][C][C][C][C][C][G]
   ← ← ← ← 절 벽 ← ← ← ← ←
```

- **S** = 시작점 (Start, 왼쪽 하단)
- **G** = 목표점 (Goal, 오른쪽 하단)
- **C** = 절벽 (Cliff) — 이곳을 밟으면 -100점의 보상을 받고 다시 시작점으로 돌아갑니다!
- 그 외 모든 칸 = 이동 시 -1점의 보상

## 우리 코드의 결과

SARSA를 500회 에피소드 동안 훈련시킨 결과입니다.

| 결과 | 수치 |
|--------|-------|
| 마지막 50회 에피소드 평균 보상 | **-21.6** |
| 최적의 (하지만 위험한) 경로 보상 | -13 |

SARSA가 학습한 정책은 **그리드의 맨 윗길**을 따라갑니다 — 안전한 우회로를 택한 것이죠! 최적 경로(-13)보다 몇 걸음 더 걸리긴 하지만(-21.6), 훈련 중에 절벽으로 떨어지는 일이 거의 없습니다.

## 실생활 예시

- **약물을 투여하는 간호사**: 조금 더 빠른 방법이 있더라도 검증된 안전한 절차(안전한 경로)를 따릅니다. 작은 실수(탐험)가 치명적일 수 있기 때문입니다.
- **항공기 조종사**: 지름길이 더 빨라 보이더라도 엄격한 체크리스트(안전한 경로)를 준수합니다. 인간의 실수를 고려한 선택입니다.
- **요리 배우기**: 위험한 요령을 부리기보다는 잘 검증된 레시피(안전)부터 시작합니다.

## 기억해야 할 주요 용어

- **온-폴리시 (On-policy)**: 실제로 사용하고 있는 정책(무작위 실수 포함)에 대해 직접 학습하는 방식
- **SARSA 업데이트**: 이론상의 최선이 아닌, *실제* 다음 행동을 사용함
- **안전한 경로**: 탐험 중의 실수를 고려하여 위험을 피하는 조금 더 긴 경로
- **TD (Temporal Difference) 제어**: 에피소드가 끝날 때까지 기다리지 않고 매 단계 가치를 업데이트함

핵심 아이디어: **SARSA는 정직합니다. 자신이 되고 싶은 모습이 아니라, 실제로 하고 있는 행동으로부터 배웁니다. 이 점이 위험한 곳 근처에서 SARSA를 조심스럽고 안전하게 만듭니다!**
