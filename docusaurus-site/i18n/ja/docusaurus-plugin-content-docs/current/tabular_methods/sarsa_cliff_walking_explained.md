# Cliff Walking 用の SARSA 🏔️

## それは何？

一方の端に **恐ろしい崖** がある **とても長い廊下** を想像してください。もし崖から落ちたら、スタート地点まで戻らなければなりません！あなたの目標は、一方の端からもう一方の端まで、崖に落ちずにできるだけ早く歩くことです。

**SARSA** は、練習することによってこの廊下の歩き方を学習するロボットです。SARSA は、たとえ少し遠回りになっても、崖を避ける **安全な経路** を通ることを学習します。なぜなら、探索中に崖の近くで誤って滑ってしまう可能性があることを知っているからです！

---

## 大きなアイデア：実際の行動から学ぶ

SARSA は以下の略です：
**S**tate (状態) → **A**ction (行動) → **R**eward (報酬) → **S**tate (状態) → **A**ction (行動)

これらは、SARSA が学習に使用する5つの情報です：

1. **S** — 今どこにいるか？（現在の状態）
2. **A** — 実際にどの行動をとったか？
3. **R** — どのような報酬を得たか？
4. **S** — どこにたどり着いたか？
5. **A** — *次に実際にどの行動をとるか*？

最後の「A」が SARSA を特別なものにしています！SARSA は、理想的な完璧な行動ではなく、*次に実際に自分がとる行動*（たとえそれが探索のためのランダムな動きであっても）を使って更新を行います。

**現実世界の例：** 自転車の乗り方を練習しているところを想像してください。自分が時々ランダムにふらつく（探索）ことを知っていれば、駐車している車から少し離れた場所を走ります。なぜなら、ふらついたときに車にぶつかってしまうかもしれないからです！SARSA も同じことをします。自分自身のランダムなミスを考慮に入れて、安全な経路を学習するのです。

## Cliff Walking (崖歩き) のマップ

```
[ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ]
[ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ]
[ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ]
[S][C][C][C][C][C][C][C][C][C][C][G]
   ← ← ← ← 崖 (CLIFF) ← ← ← ← ←
```

- **S** = スタート (左下)
- **G** = ゴール (右下)
- **C** = 崖 (Cliff) — ここを踏むと報酬 -100、最初からやり直し！
- その他のステップ = 報酬 -1

## コードから分かったこと

SARSA を 500 エピソード訓練した後の結果：

| 結果 | 値 |
|--------|-------|
| 最後の 50 エピソードの平均報酬 | **-21.6** |
| 最適な（リスクのある）経路の報酬 | -13 |

SARSA が学習した方策は、**グリッドの一番上**を通るルートです。これは安全な回り道です！最短ルート（-13）に比べて数歩余計にかかります（-21）が、訓練中に崖から落ちることはほとんどありません。

## 現実世界の例

- **薬を投与する看護師**：わずかに早い方法があったとしても、立証された安全な手順（安全な経路）に従います。なぜなら、小さなミス（探索）が危険につながる可能性があるからです。
- **飛行機のパイロット**：近道の方が早く見える場合でも、ヒューマンエラーを考慮して、厳格なチェックリスト（安全な経路）に従います。
- **料理の習得**：リスクのある近道をするのではなく、十分に検証されたレシピ（安全）から始めます。

## 覚えておくべきキーワード

- **オンポリス (On-policy)**：実際に使用している方策（ランダムなミスを含む）について学習する
- **SARSA の更新**：理論上の最善ではなく、*実際*の次の行動を使用する
- **安全な経路**：探索によるミスを考慮し、危険を避ける長めの経路
- **TD (時間的差分) 制御**：エピソードの終了を待たず、1ステップごとに値を更新する

大きなアイデア： **SARSA は正直です。理想的な自分ではなく、実際の行動から学習します。そのため、危険の近くでは慎重で安全な動きになります！**
