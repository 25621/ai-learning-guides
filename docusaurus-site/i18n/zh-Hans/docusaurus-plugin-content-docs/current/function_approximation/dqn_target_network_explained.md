# 目标网络 (Target Network)：稳定靶心 🎯

## 移动的球门问题

想象一下你正试图用弓箭射中靶心。你射出一箭，看看箭落在哪，然后为下一次调整瞄准。很简单，对吧？

现在想象一下，每次你射箭时，靶心都会移动！你射出的每一支箭都会稍微改变下一次射击时靶心的位置。你永远无法进步——你一直在追逐一个不断跑掉的目标。

这就是没有目标网络的 DQN 面临的问题！

---

## 为什么 Q 目标值会不断移动

在 DQN 中，每次更新的目标值 (Target) 是：
> target = reward + γ × max(Q(next_state))

这里的 **γ (gamma)** 是 **折扣因子 (discount factor)** —— 一个介于 0 和 1 之间的数字（通常为 0.99），它控制着智能体对 *未来* 奖励与 *即时* 奖励的重视程度。

**现实生活中的例子：** 想象有人给你两个选择：现在给你一块饼干，或者明天给你两块饼干。如果你非常想现在吃饼干，你的 γ 就很低（你对未来进行了大幅折扣）。如果你很有耐心，乐于等待，你的 γ 就很高（未来的奖励几乎和现在一样重要）。在强化学习中，γ = 0.99 意味着“下一步的奖励价值相当于现在的 99%”。

等式右侧的 Q 值来自于……我们正在训练的同一个网络！

因此，每当我们更新网络（为了让 Q 值更好）时，我们也改变了目标值。这是一个反馈循环：

1. 更新网络 → Q 值改变
2. Q 值改变 → 目标值改变
3. 目标值改变 → 以不同的方式更新网络
4. 永远循环下去 —— 不稳定！

**现实生活中的例子：** 试着在一部每次你踩上去都会改变读数的体重秤上称重。你永远无法知道你的真实体重！

---

## 解决方案：冻结靶心！❄️

**目标网络 (Target Network)** 是主 Q 网络的一个副本，它被冻结在原处。

- **在线网络** (`qnet`)：每个训练步骤都会更新 —— 学习速度快
- **目标网络** (`target_net`)：冻结的副本 —— 每 100 步才更新一次

我们使用冻结的目标网络来计算目标值：
> target = reward + γ × max(Q_TARGET(next_state))

目标值在 100 步内保持不动！这给在线网络提供了一个稳定的目标去瞄准。然后我们将在线网络的权重复制到目标网络中，再次冻结，周而复始。

**现实生活中的例子：** 想想学生和老师。老师布置家庭作业（目标）。学生学习并进步。经过 100 节课后，老师更新家庭作业，使其变得更难。老师不会每分钟都变——那太混乱了！

---

## 完整的 DQN 配方 🍕

完整的 DQN 算法（经验回放 + 目标网络）如下：

```
1. 初始化在线网络 Q 和目标网络 Q_target（具有相同的权重）
2. 创建回放缓冲区（记忆盒）

每一个环境步骤：
  a. 使用基于 Q 的 ε-贪婪策略选择动作
  b. 将 (状态, 动作, 奖励, 下一状态) 存入缓冲区

每 4 个步骤：
  c. 从缓冲区中抽取随机的小批量数据
  d. 使用 Q_TARGET 计算目标值（已冻结！）
  e. 更新 Q 以最小化损失 (loss)

每 100 个步骤：
  f. 将 Q 的权重复制给 Q_TARGET（同步目标网络）
```

这正是 DeepMind 2015 年 DQN 论文中的算法！

## 比较结果显示了什么

当你运行 `dqn_target_network.py` 时，你会看到：

**没有目标网络（仅使用 DQN + 经验回放）：**
- 训练可能“还可以”，但会周期性崩溃
- Q 值可能会发散（爆炸或震荡）
- 学习过程较难预测

**完整 DQN（经验回放 + 目标网络）：**
- 学习曲线更稳定地上升
- Q 值保持在合理范围内
- 更快地收敛到解决阈值（CartPole 任务为 195+ 分）

## “致命三元组” (The Deadly Triad) ☠️

在强化学习中，结合以下三件事会产生不稳定性：

1. **函数近似 (Function approximation)**（使用神经网络代替表格） ← 我们使用了这个
2. **自举 (Bootstrapping)**（使用 Q 值来估计 Q 值） ← 我们使用了这个
3. **异策略学习 (Off-policy learning)**（Q 学习使用 max 操作，而非实际策略） ← 我们使用了这个

这三者结合 = “致命三元组”。DQN 通过以下方式驯服了它：
- 经验回放 → 打破相关性
- 目标网络 → 打破反馈循环

它并没有完全解决问题，但使其变得可控！

## 关键词汇

| 单词 | 含义 |
|------|---------|
| **目标网络 (Target Network)** | Q 网络的一个冻结副本，仅用于计算目标值 |
| **在线网络 (Online Network)** | 正在被积极训练的 Q 网络 |
| **同步 (Sync)** | 将在线网络的权重复制到目标网络中 |
| **反馈循环 (Feedback Loop)** | 当系统的输出反馈回来改变输入时（可能导致不稳定） |
| **致命三元组 (Deadly Triad)** | 函数近似 + 自举 + 异策略学习的组合，会导致不稳定性 |
