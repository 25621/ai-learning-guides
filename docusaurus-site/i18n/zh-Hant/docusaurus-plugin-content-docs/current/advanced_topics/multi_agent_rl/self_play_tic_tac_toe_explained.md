# 自博弈 (Self-Play)：透過讓智能體自己與自己玩來教學 ♟️

## 什麼是自博弈？

想像一個想要精通國際象棋的孩子，但她找不到對手。於是她自己和自己玩。左手對右手。每一局，*雙方*都努力求勝。每一局，*雙方*都從中學習成功的經驗。

這就是**自博弈 (Self-Play)**：同一個智能體同時扮演兩名玩家，每一次移動都成了對下一位移動者的教訓。沒有老師，沒有專家對手。只有一個學習者，而它本身就是自己不斷向上攀登的階梯。

自博弈聽起來像個小花招 —— 難道不需要真正的對手嗎？—— 但它實際上是過去十年中許多著名強化學習里程碑背後的引擎：**AlphaGo Zero**、**AlphaZero**、**MuZero**、**OpenAI Five**。它們都使用了自博弈。原因很簡單：隨著你的進步，你的對手也以同樣的速度在進步。挑戰始終與你的技能水平相匹配。

---

## 為什麼它有效

有三點讓自博弈變得特別：

1. **無窮無盡的對手。** 你永遠不會缺對手。對手隨時隨地都在，而且是免費的。
2. **隨你共同成長的課程。** 初學者只能和初學者玩。隨著你變強，你的「影子」也會自動變強。
3. **對稱性。** 在零和遊戲（一方的勝利就是另一方的失敗）中，一套 Q 值可以描述雙方；你只需要在輪到另一方時翻轉符號即可。因此，*單一*的 Q 表就可以實現自我教學。

井字棋 (Tic-tac-toe) 是完美的測試場：它足夠小，可以用字典存儲，但也足夠複雜，面對有策略的玩家，隨機選擇步法幾乎總是會導致失敗。

## 現實生活中的類比

- **對著牆練習網球。** 你不會輸給牆，但你可以練習發球。自博弈就像是在兩端都練習 —— 你既是牆又是球員，並在兩者之間來回切換。
- **辯論俱樂部練習。** 那些總是嘗試站在自己觀點的對立面進行辯護的人，往往會成為更優秀的辯論者。每一次辯論都在同時訓練進攻和防守。
- **AlphaGo Zero。** 它在學習過程中沒有使用任何人類對局數據。它從隨機移動開始，與自己進行了數百萬次對局；在短短幾天內，它就超越了之前所有的圍棋程序，包括擊敗了李世石的那一個版本。

## 我們的代碼做了什麼

我們為*當前輪到移動的玩家*學習一個 Q 表：

```
Q[(棋盤狀態, 輪到誰走)][動作] = 該玩家的預期回報
```

訓練循環如下：

1. 從空棋盤開始。`player = X`。
2. 雙方玩家都由**同一個智能體**扮演，使用 ε-貪婪策略。
3. 每局遊戲結束後，回溯歷史記錄中的每一個（棋盤、玩家、動作）三元組，並應用 Q-learning 更新。
4. 獎勵在不同回合間正負號翻轉：如果 X 獲勝，X 做出的每一步都得到 +1（或從未來的獲勝狀態引導價值）；O 做出的每一步得到 -1。
5. 我們慢慢降低（衰減）我們的探索率 (ε)，從 0.2 → 0.02，以便智能體在訓練後期致力於其最佳表現，而不是嘗試隨機動作。

每隔 2,500 個回合，我們就會讓智能體與一個**隨機對手**進行評估（我們凍結學習過程，評估期間不對 Q 表進行更新，且雙方都採取貪婪策略）。在足夠的自博弈之後，智能體應該能贏得或平掉約 100% 的對局。

### 你應該看到的結果

在進行 50,000 次自博弈回合後：

| 對陣雙方 | 預期結果 |
|----------|-----------------|
| 訓練後的智能體 vs 隨機對手 (1000 局) | **~95-99% 勝或平**，幾乎 0% 失敗 |
| 訓練後的智能體 vs 自身 (200 局貪婪對弈) | **全部 200 局平局**。井字棋是一個如果雙方都玩得完美，最後總會以平局告終遊戲。自博弈產生全是平局的結果，正是收斂的標誌。 |

## 注意這些細節

- **符號翻轉至關重要。** 一個常見的錯誤是：忘記了「對手最大化他們的價值」意味著在目標計算中「最小化我們的價值」。我們代碼中的更新使用的是 `target = reward - gamma * max(Q[下一個狀態, 對手])`。
- **此處未利用對稱性。** 真正的實現會對棋盤進行規範化（意味著它們會將任何棋盤狀態旋轉或鏡像為標準的、唯一的「規範形式」，以便智能體識別相同的局勢），從而在 8 種對稱性中共享 Q 值。我們跳過這一步 —— 狀態空間足夠小，可以進行暴力計算。
- **Q 表的增長。** 在 50k 次自博弈後，你會看到幾千個「狀態-玩家」鍵值對。這在這裡沒問題；但對於國際象棋或圍棋，你需要神經網絡來代替。這就是為什麼 **AlphaZero 用 CNN + MCTS 替換了表格**。

## 自博弈在哪些情況下會失效

- **非零和博弈。** 「雙贏」與對稱博弈是不相容的；你不能簡單地翻轉正負號。
- **非對稱角色。** 如果「進攻方」和「防守方」的動作空間不同，你需要兩個獨立的網絡。
- **策略循環。** 純粹的自博弈可能會陷入類似「石頭剪刀布」的循環中。AlphaStar 透過維護一個巨大的過往版本智能體池（或稱「聯賽」），並從中隨機抽取對手來解決這個問題，從而使智能體學習擊敗多種不同的遊戲風格，而不只是當前的這一種。
- **獎勵劫持 (Reward hacking)。** 自博弈會讓雙方變聰明，但僅限於你*定義的*遊戲。如果你的獎勵系統有漏洞（例如獎勵玩家存活更久而不是獲勝），雙方會共同利用該漏洞，導致產生怪異且無用的行為，而不是精通真正的遊戲。

## 需要記住的核心詞彙

| 詞彙 | 含義 |
|------|---------|
| **自博弈 (Self-play)** | 同一個智能體扮演遊戲的兩方 |
| **零和 (Zero-sum)** | 一方的收益 = 另一方的損失 |
| **對稱性 (Symmetry)** | 如果翻轉正負號，一個 Q 表可以服務於雙方 |
| **種群博弈 (Population play)**| 與*許多*過去版本的自己作為對手進行自博弈 (AlphaStar) |
| **課程學習 (Curriculum)** | 自然的難度遞進 —— 自博弈可以免費獲得這一點 |
| **MCTS** | 蒙地卡羅樹搜尋 —— AlphaZero 與自博弈配合使用的規劃算法 |

---

## 一句話總結

> **自博弈將進步變成了它自己的階梯：每當你變強，你的對手也會自動變强。**

這一想法在結合了**神經網絡**（受大腦啟發的從數據中學習模式的數學函數）和樹搜尋後，在圍棋、象棋、將棋、Dota 2 和星際爭霸中擊敗了人類頂尖高手。
