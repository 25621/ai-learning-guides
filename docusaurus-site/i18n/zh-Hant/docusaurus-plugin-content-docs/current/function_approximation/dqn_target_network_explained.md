# 目標網路 (Target Network)：穩定靶心 🎯

## 移動的球門問題

想像一下你正試圖用弓箭射中靶心。你射出一箭，看看箭落在哪，然後為下一次調整瞄準。很簡單，對吧？

現在想像一下，每次你射箭時，靶心都會移動！你射出的每一支箭都會稍微改變下一次射擊時靶心的位置。你永遠無法進步——你一直在追逐一個不斷跑掉的目標。

這就是沒有目標網路的 DQN 面臨的問題！

---

## 為什麼 Q 目標值會不斷移動

在 DQN 中，每次更新的目標值 (Target) 是：
> target = reward + γ × max(Q(next_state))

這裡的 **γ (gamma)** 是 **折扣因子 (discount factor)** —— 一個介於 0 和 1 之間的數字（通常為 0.99），它控制著代理人對 *未來* 獎勵與 *即時* 獎勵的重視程度。

**現實生活中的例子：** 想像有人給你兩個選擇：現在給你一塊餅乾，或者明天給你兩塊餅乾。如果你非常想現在吃餅乾，你的 γ 就很低（你對未來進行了大幅折扣）。如果你很有耐心，樂於等待，你的 γ 就很高（未來的獎勵幾乎和現在一樣重要）。在強化學習中，γ = 0.99 意味著「下一步的獎勵價值相當於現在的 99%」。

等式右側的 Q 值來自於……我們正在訓練的同一個網路！

因此，每當我們更新網路（為了讓 Q 值更好）時，我們也改變了目標值。這是一個回饋循環：

1. 更新網路 → Q 值改變
2. Q 值改變 → 目標值改變
3. 目標值改變 → 以不同的方式更新網路
4. 永遠循環下去 —— 不穩定！

**現實生活中的例子：** 試著在一部每次你踩上去都會改變讀數的體重計上稱重。你永遠無法知道你的真實體重！

---

## 解決方案：凍結靶心！❄️

**目標網路 (Target Network)** 是主 Q 網路的一個副本，它被凍結在原處。

- **在線網路** (`qnet`)：每個訓練步驟都會更新 —— 學習速度快
- **目標網路** (`target_net`)：凍結的副本 —— 每 100 步才更新一次

我們使用凍結的目標網路來計算目標值：
> target = reward + γ × max(Q_TARGET(next_state))

目標值在 100 步內保持不動！這給在線網路提供了一個穩定的目標去瞄準。然後我們將在線網路的權重複製到目標網路中，再次凍結，周而復始。

**現實生活中的例子：** 想想學生和老師。老師布置家庭作業（目標）。學生學習並進步。經過 100 節課後，老師更新家庭作業，使其變得更難。老師不會每分鐘都變——那太混亂了！

---

## 完整的 DQN 配方 🍕

完整的 DQN 演算法（經驗回放 + 目標網路）如下：

```
1. 初始化在線網路 Q 和目標網路 Q_target（具有相同的權重）
2. 建立回放緩衝區（記憶盒）

每一個環境步驟：
  a. 使用基於 Q 的 ε-貪婪策略選擇動作
  b. 將 (狀態, 動作, 獎勵, 下一狀態) 存入緩衝區

每 4 個步驟：
  c. 從緩衝區中抽取隨機的小批次資料
  d. 使用 Q_TARGET 計算目標值（已凍結！）
  e. 更新 Q 以最小化損失 (loss)

每 100 個步驟：
  f. 將 Q 的權重複製給 Q_TARGET（同步目標網路）
```

這正是 DeepMind 2015 年 DQN 論文中的演算法！

## 比較結果顯示了什麼

當你執行 `dqn_target_network.py` 時，你會看到：

**沒有目標網路（僅使用 DQN + 經驗回放）：**
- 訓練可能「還可以」，但會週期性崩潰
- Q 值可能會發散（爆炸或震盪）
- 學習過程較難預測

**完整 DQN（經驗回放 + 目標網路）：**
- 學習曲線更穩定地上升
- Q 值保持在合理範圍內
- 更快地收斂到解決閾值（CartPole 任務為 195+ 分）

## 「致命三元組」 (The Deadly Triad) ☠️

在強化學習中，結合以下三件事會產生不穩定性：

1. **函數近似 (Function approximation)**（使用神經網路代替表格） ← 我們使用了這個
2. **自舉 (Bootstrapping)**（使用 Q 值來估計 Q 值） ← 我們使用了這個
3. **異策略學習 (Off-policy learning)**（Q 學習使用 max 操作，而非實際策略） ← 我們使用了這個

這三者結合 = 「致命三元組」。DQN 透過以下方式馴服了它：
- 經驗回放 → 打破相關性
- 目標網路 → 打破回饋循環

它並沒有完全解決問題，但使其變得可控！

## 關鍵詞彙

| 單字 | 含義 |
|------|---------|
| **目標網路 (Target Network)** | Q 網路的一個凍結副本，僅用於計算目標值 |
| **在線網路 (Online Network)** | 正在被積極訓練的 Q 網路 |
| **同步 (Sync)** | 將在線網路的權重複製到目標網路中 |
| **回饋循環 (Feedback Loop)** | 當系統的輸出回饋回來改變輸入時（可能導致不穩定） |
| **致命三元組 (Deadly Triad)** | 函數近似 + 自舉 + 異策略學習的組合，會導致不穩定性 |
